Pyhton Script

Python Automation Script

This Python Automation Script will readPDF's out loud for you.

Installation
#1
Check whether you are running latest package manager pip
pip install --upgrade pip
#2
Use the package manager to install pyttsx3.
pip install pyttsx3
#3
Similarly install PyPDF2.
pip install PyPDF2 library
Usage

Copy & Paste your desired PDF in project folder.

#4
Import installed libraries
import pyttsx3
import PyPDF2
#5
Open the book you want to listen
book = open('pdf_name.pdf', 'rb')
Reads Book
pdfReader = PyPDF2.PdfFileReader(book)
#6
Reading total no. pages from book
Calculates total number of pages from book
pages = pdfReader.numPages
Returns Total Number of pages
print(pages)
#7
Initialize Python Text to Speech
bot = pyttsx3.init()
#8
Start Speaking
for num in range(0, pages):
page = pdfReader.getPage(0)
text = page.extractText()
bot.say(text)
bot.runAndWait()
